#Gestionnaire Hôspitalier

## Nom du projet et du groupe
**Application-Hopital**
Crée par VOUSEMER Doryan, TANOH Laetitia, LOKO Marie-Paule et NICAUD Rosalie

## Description métier
Cette application est un gestionnaire hospitalier développé en Java.
L'application permet de gérer le personnel medical, les patients, les urgences, les salles, les soins, les soins, les consultations, les urgences ainsi que les actes chirurgicaux.
Le système repose sur une architecture orientée objet avec des classes abstraites, interfaces et generics.

## Technologie d'interface
**Option B - Web (Servlet/JSP) avec Apache Tomcat**
Nous avons choisi l’option B du cahier des charges,une application web en Servlet/JSP, car cette technologie nous semblait plus familière et plus adaptée pour créer une interface accessible et organisée.

## Liste des fonctionnalités
- Gestion des patients : dossier médical, antécédents, facturation
- Gestion du personnel médical : médecins avec spécialités et plannings
- Gestion des soins : consultations et actes chirurgicaux
- Gestion des salles et lits : occupation et disponibilité
- File d'attente aux urgences avec niveau de priorité
- Statistiques dynamiques sur le tableau de bord
